Finatra 2.5.0 has been released to Maven Central

posted by cacoco on 2016-10-13 09:36:43 -0700

Finatra release 2.5.0 is now available from Maven Central.

Some of the changes in the latest releases:

  • We’ve upgraded to Jackson 2.8.3!
  • We’ve mostly removed the need of the maven.twttr.com sbt resolver. It’s still necessary to resolve specific Twitter thrift dependencies.
  • We fixed an issue with missing files in the finatra-jackson test jar. Classes in the test c.t.finatra.validation package were not properly marked for inclusion.
  • SLFJ4 bridge installation has changed to occur earlier and capture more of the bridged logging.
  • Simplified ExceptionMapper usage. Please read the commit notes as this is a breaking change.
  • We’ve added support for “dark traffic filtering” in finatra-http.

Please checkout the finatra-2.5.0 release notes on github for more information.

See the Finatra Github page or checkout our User Guide for more information on getting started.
